How to Use The App

Step 1: Install the metamask plugin plugin and create an account on Ropsten Test Network and load some Ether.
Step 2: Purchase tokens in User Section by entering the total number of tokens you like to buy.
Step 3: Submit the Bug Report from the User Section using the Tokens.
Step 4: If your Vulnerability is accepted with 4 acceptance votes you get the bounty tokens worth 5 times your bet.
Step 5: Enter your account address to look up your activity.
Step 6: If you want to be a part of the security team and validate the Bugs, email your public key to girishkathirdy@gmail.com.
